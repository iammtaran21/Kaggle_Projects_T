# Heart_Disease_XGB
- Here we have a data set and we have to predict and classify.
- Based on the given Health parameters such as Age,Cholestrol, etc. , we need to predict if a person is prone to Heart Disease or not.
- I have used XGB Classification Model.
- Accuracy score is 97% 
- The dataset is available at https://www.kaggle.com/ronitf/heart-disease-uci
